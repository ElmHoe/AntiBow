# AntiBow
A simple plugin that allows you to block the use of bows.
Supports as many regions as needed, and works in-game.

## Versions:
* [Spigot 1.8.8](https://drive.google.com/open?id=0B-wfHJu6Mm0cZ1N3UmxfRXZIYzQ)
